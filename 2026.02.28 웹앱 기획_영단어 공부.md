### 목적
- 다음과 같은 기능을 제공하는 웹앱을 개발한다.
	- AI를 활용하여 영어 단어 및 표현의 의미와 예문을 확인한다.
	- 확인한 단어 및 표현의 의미와 예문을 구글 시트에 저장한다.
	- 구글 시트에 저장한 내용으로 퀴즈를 제공한다.

### 주요 기능
##### 구글 계정 로그인(SSO)
- 구글 계정으로 로그인하면 설정을 저장할 수 있고 과거에 저장된 설정을 그대로 사용할 수 있다.
##### 설정 항목
- gemini API key
- 구글 시트 파일 url
- 위 두 개의 설정항목에 대해 입력하고 저장하도록 한다. 
##### Gemini를 통한 영단어 및 표현 검색
- 설정된 gemini API key를 사용하여 gemini에 질문할 수 있다.
- 질문 내용은 고정된 프롬프트를 사용한다. 이용자는 오로지 영어 단어 및 표현 또는 문장을 입력한다. 
- Gemini의 답변을 화면에 노출한다.
- Gemini의 답변 중 구글 시트 저장용 데이터는 별도로 표시한다(아래에서 상세 설명).
##### 결과 저장
- Gemini의 답변에서 구글 시트 입력용으로 도출된 결과는 항목별로 선택하여 저장할 수 있게 한다.
- 이 데이터의 컬럼은 "날짜, 단어 및 표현, 뜻, 예문"으로 구성되어 있다.
- 한 단어 또는 표현에 대해 위 항목을 하나의 세트로 다루고, 세트별로 선택한 항목만 저장하도롣 한다.
- 저장하는 대상 구글 시트는 설정에서 url을 입렫한 파일이다.
- 해당 파일에 이미 값이 있는 행의 다음 행에 데이터를 입력한다.
- 데이터를 입력할 때에는 헤더 정보는 입력하지 않도록 한다.
##### 퀴즈
- 구글 시트에 저장된 데이터를 바탕으로 이용자에게 퀴즈를 제시한다.
- 퀴즈 제시도 gemini에게 고정된 프롬프트로 요청하고 gemini가 제공하는 퀴즈 양식을 웹앱에 노출한다.

### 프롬프트
- 각 기능별로 gemini에게 다음과 같은 프롬프트로 요청한다.
- 단어, 표현, 문장 검색을 위한 프롬프트

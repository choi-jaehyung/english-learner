Lexi — Privacy Policy

Last updated: 2026-05-03 · English version below

한국어

1. 개요

Lexi("본 앱")는 영어와 일본어 단어 학습을 도와주는 모바일 애플리케이션입니다. 본 개인정보 처리방침은 본 앱이 사용자 정보를 어떻게 다루는지 설명합니다.

2. 수집하는 정보

본 앱은 어떠한 사용자 데이터도 수집·저장·전송하지 않습니다.

본 앱은 별도의 백엔드 서버를 운영하지 않으며, 사용자 분석 도구(Google Analytics, Firebase 등)를 사용하지 않습니다.

3. 사용자 기기에 저장되는 정보

본 앱은 사용자가 직접 입력·생성한 다음 정보를 사용자 본인의 기기에만 저장합니다:

  • 검색한 단어와 학습 데이터: SQLite 데이터베이스 형태로 앱 내부 저장소에 저장됩니다.
  • Gemini API 키: 사용자가 등록한 Google Gemini API 키는 iOS Keychain(SecureStore)에 암호화되어 저장됩니다.
  • 앱 설정: 활성 언어, TTS 음성·속도 등 사용자 설정.

4. iCloud 동기화

사용자가 iCloud Drive를 활성화한 경우, 본 앱은 사용자 본인의 iCloud Drive 컨테이너에 단어 데이터(SQLite 파일)를 저장하여 같은 Apple ID로 로그인된 다른 기기와 동기화합니다.

  • 이 데이터는 사용자 본인의 Apple ID 소유이며, Apple과 사용자 간의 약정에 따라 처리됩니다.
  • 본 앱 개발자는 이 데이터에 접근할 수 없습니다.
  • 사용자가 Files 앱에서 iCloud Drive → Lexi 폴더에서 직접 데이터 파일을 확인·관리할 수 있습니다.

5. 외부 서비스 — Google Gemini API

본 앱의 단어 검색 및 퀴즈 생성 기능은 사용자가 등록한 본인의 Gemini API 키를 통해 Google의 Gemini API에 직접 호출됩니다.

  • 검색하신 텍스트는 사용자의 기기에서 직접 Google 서버로 전송되며, 본 앱 개발자의 서버를 거치지 않습니다.
  • Google의 데이터 처리 방침은 Google 개인정보 처리방침을 따릅니다.
  • 사용자는 언제든지 Settings 화면에서 자신의 API 키를 삭제하여 Google과의 연결을 해제할 수 있습니다.

6. 음성 합성(TTS)

발음 듣기 기능은 iOS 기기 자체의 음성 합성 엔진(AVSpeechSynthesizer)을 사용하며, 외부 서버에 어떠한 데이터도 전송하지 않습니다.

7. 어린이의 개인정보 보호

본 앱은 4세 이상 모두 사용 가능하지만, 만 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다(앱 자체가 어떠한 개인정보도 수집하지 않으므로 적용 불가).

8. 데이터 삭제

  • 앱 내부 데이터: Lexi의 Settings → "저장된 단어 데이터 초기화"를 통해 삭제 가능
  • API 키: Settings → "API 키 전체 삭제"를 통해 삭제 가능
  • iCloud 데이터: Files 앱 → iCloud Drive → Lexi 폴더에서 직접 삭제
  • 앱 자체: iOS의 표준 앱 삭제 절차를 통해 모든 로컬 데이터가 함께 제거됩니다.

9. 정책 변경

본 정책이 변경되는 경우, 최신 일자를 본 페이지 상단에 명시합니다.

English

1. Overview

Lexi ("the App") is a mobile application that helps users learn English and Japanese vocabulary. This Privacy Policy explains how the App handles user information.

2. Information We Collect

The App does not collect, store, or transmit any user data.

The App does not operate any backend server and does not use analytics services (such as Google Analytics or Firebase).

3. Information Stored on Your Device

The App stores the following information that you create or enter, only on your own device:

  • Searched words and study data: Stored as a SQLite database within the App's local storage.
  • Gemini API key: The Google Gemini API key you register is stored encrypted in iOS Keychain (SecureStore).
  • App settings: Active languages, TTS voice and speed preferences.

4. iCloud Sync

If you have iCloud Drive enabled, the App stores your word data (SQLite file) in your own iCloud Drive container, syncing with other devices signed in with the same Apple ID.

  • This data belongs to your Apple ID and is governed by the terms between you and Apple.
  • The App developer cannot access this data.
  • You can directly view and manage the data file via the Files app → iCloud Drive → Lexi folder.

5. Third-Party Service — Google Gemini API

The word search and quiz generation features call the Google Gemini API directly using your registered API key.

  • Text you search is sent directly from your device to Google's servers, never passing through any developer-controlled server.
  • Google's data handling is governed by the Google Privacy Policy.
  • You can disconnect from Google at any time by removing your API key in the Settings screen.

6. Text-to-Speech (TTS)

The pronunciation feature uses the device's built-in speech synthesis engine (AVSpeechSynthesizer) and sends no data to any external server.

7. Children's Privacy

The App is rated 4+ for all ages but does not knowingly collect personal information from children under 13 (the App does not collect any personal information at all, so this is non-applicable).

8. Data Deletion

  • In-app data: Delete via Settings → "저장된 단어 데이터 초기화"
  • API keys: Delete via Settings → "API 키 전체 삭제"
  • iCloud data: Delete directly via Files app → iCloud Drive → Lexi folder
  • The App itself: All local data is removed upon standard iOS app uninstall.

9. Policy Changes

If this policy changes, we will update the date at the top of this page.
